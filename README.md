
# Portfolio Manager

This Java application demonstrates object-oriented programming concepts using financial assets.

## Features
- BankAccount class
- Valuable interface
- Abstract FixedAsset class
- Gold, Jewelry, and House asset classes
- Demonstrates inheritance and polymorphism

## Concepts Practiced
- Interfaces
- Abstract classes
- Inheritance
- Method overriding
- Constructors
- Packages
- Polymorphism
- Collections (ArrayList)
